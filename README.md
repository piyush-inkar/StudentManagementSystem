# Student Management System Project 

The Student Management System is a console-based application, implemented using Core Java.
The objective of this project is to provide a facility for managing students' information using a menu-based interface.

The application enables users to add, view, search, and delete students' information, etc. 
This application is based on Object-Oriented Programming (OOP) Concepts, Project Structure, and basic data management using Java collections.

## Features

- Add new student records

- View all students

- Search student by ID

- Delete student record

- Menu-driven console interface

- Organized project structure using packages

## Technologies Used
- Java (Core Java)

- IntelliJ IDEA

- Java Collections (ArrayList)

- Object-Oriented Programming

## Future Improvements

- Update student information

- Prevent duplicate student IDs

- File handling to store student data

- Exception handling for better input validation

- Sorting students by name or ID
